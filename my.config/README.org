** rules
1. 本目录下的配置文件中，起始行标示了如何使用该文件的部分信息
2. bash和zsh的代理设置将根据当前登陆的用户名来决定
   但目前其他软件的代理设置只能手动修改
3. bashrc.sh将作为所有shell的默认配置文件，被各shell主动加载
