   安装linux字体
1. .ttf格式的文件拷贝至/usr/share/fonts/中的任意目录下, 推荐创建一个新目录来存放, e.g. /usr/share/fonts/tony
2. 切换至放置新字体的目录下, 执行以下命令
cd /usr/share/fonts/tony
sudo mkfontscale
sudo mkfontdir
sudo fc-cache -fv
1. fc-list就能看到当前安装的所有字体信息了
